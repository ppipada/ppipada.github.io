    Longest palindromic substring

    Dynamic-Programming
    String
    Level-4
    Levels: level-4
    Data structures: string
    Algorithms: dynamic-programming

    Practice Link: LeetCode

    Description:

    • Given a string s, find the longest palindromic substring in s.
    • You may assume that the maximum length of s is 1000.

    Example 1:

    1Input: "babad"
2Output: "bab"
3Note: "aba" is also a valid answer.

    Example 2:

    1Input: "cbbd"
2Output: "bb"

    Python Solution:

     1class Solution(object):
 2    def longestPalindrome(self, s):
 3        """
 4        :type s: str
 5        :rtype: str
 6        """
 7
 8        res = ""
 9        for i in range(len(s)):
10            # odd case, like "aba"
11            tmp = self.helper(s, i, i)
12            if len(tmp) > len(res):
13                res = tmp
14            # even case, like "abba"
15            tmp = self.helper(s, i, i + 1)
16            if len(tmp) > len(res):
17                res = tmp
18        return res
19
20    # get the longest palindrome, l, r are the middle indexes
21    # from inner to outer
22    def helper(self, s, l, r):
23        while l >= 0 and r < len(s) and s[l] == s[r]:
24            l -= 1
25            r += 1
26        return s[l + 1:r]

    Meeting rooms

    Array
    Level-4
    Intervals
    Levels: level-4
    Data structures: array
    Patterns: intervals

    Description:

    • Given an array of meeting time intervals consisting of start and end times [[s1,e1],[s2,e2],...] (si < ei), determine if a person could attend all meetings.

    Example 1:

    1Input: [[0, 30],[5, 10],[15, 20]],
2Output: false.

    Python Solution:

     1import operator
 2
 3
 4class Interval:
 5    def __init__(self, s=0, e=0):
 6        self.start = s
 7        self.end = e
 8
 9
10class Solution:
11    def canAttendMeetings(self, intervals):
12        """
13        :type intervals: list[Interval]
14        :rtype: bool
15        """
16        intervals.sort(key=operator.attrgetter("start"))
17        for i in range(len(intervals) - 1):
18            if intervals[i].end > intervals[i + 1].start:
19                return False
20
21        return True

    Meeting rooms II

    Array
    Level-4
    Intervals
    Levels: level-4
    Data structures: array
    Patterns: intervals

    Description:

    • Given an array of meeting time intervals consisting of start and end times [[s1,e1],[s2,e2],...] (si < ei), find the minimum number of conference rooms required.
    • i.e Find the maximum number of overlapped intervals

    Example 1:

    1Input: [[0, 30],[5, 10],[15, 20]],
2Output: 2

    Python Solution:

     1import heapq
 2import operator
 3
 4
 5class Interval:
 6    def __init__(self, s=0, e=0):
 7        self.start = s
 8        self.end = e
 9
10
11class Solution(object):
12    def minMeetingRooms(self, intervals):
13        """
14        :type intervals: list[Interval]
15        :rtype: int
16        """
17        maxa = 0
18
19        intervals.sort(key=operator.attrgetter("start"))
20        h_end = []
21        for itvl in intervals:
22            heapq.heappush(h_end, itvl.end)
23            while h_end and h_end[0] <= itvl.start:
24                heapq.heappop(h_end)
25
26            maxa = max(maxa, len(h_end))
27
28        return maxa
